# RagnaForge Validate

Use para validação local.

## Comando Futuro

```sh
ragnaforge validate --json
```

## Regras

- Não aplicar correções automaticamente.
- Separar erros por severidade.
- Gerar relatório objetivo.
- Validar profile/config/cache antes de confiar nos índices.
