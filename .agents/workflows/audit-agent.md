# audit-agent

Audite o Agente Setimmo e o projeto principal sem aplicar alterações.

## Passos

1. Leia `docs/AI_AGENT_CONTRACT.md`.
2. Leia `AGENTS.md`.
3. Verifique estrutura do agente.
4. Consulte docs principais do projeto RagnaForge se necessário.
5. Rode `git status` onde aplicável.
6. Rode `ragnaforge status --json`, se disponível.
7. Rode `ragnaforge doctor --json`, se disponível.
8. Verifique activeProfile e configFingerprint.
9. Produza Artifact com:
   - achados
   - riscos
   - arquivos lidos
   - paths ativos
   - fingerprint atual
   - próximos passos
   - confirmação de nenhuma alteração destrutiva
