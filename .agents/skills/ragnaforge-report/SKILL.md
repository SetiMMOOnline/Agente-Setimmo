# RagnaForge Report

Use para relatórios finais.

## Comando Futuro

```sh
ragnaforge report --operation <operation_id> --format json
```

## Regras

- Confirmar arquivos criados/alterados/lidos.
- Confirmar testes.
- Confirmar proteções.
- Confirmar activeProfile/configFingerprint.
- Confirmar ausência de alterações destrutivas.
